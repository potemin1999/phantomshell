Функции.
 
 В любом современном языке программирования/командном языке есть возможность повторно исполнить участок кода без использования циклов при помощи функций. 
 
 Создание функции без аргументов:
  def func1 {
   def variable1 = 42
  }
     или
  def func1() {
   def variable1 = 24
  }
 
 Создание функции с одним аргументом любого типа, возвращающей значение:
  def func2(arg1) {
   def variable2 = 128 + arg1;
   return variable2
  }
 
 Возможно описание типа аргументов функции:
  def func3(Integer arg1, Float arg2) {
   return arg1 + arg2
  }
 
 Передача значений аргументов, за исключением системных и пользовательских объектов, осуществляется по значению. 
 Вызов функций может происходить следующим образом:
  func1()      //обычный стиль
  func1        //командный стиль
  func2(666)   //обычный стиль
  func2 999    //командный стиль
  def floatVar = func3(1,1.0) //обычный стиль с последующим присвоением полученного значения переменной
  def floatVar2 = func3 floatVar 1
   /* командный стиль с преобразованием значения переменной floatVar типа Float к типу Integer
      и присвоением значения переменной floatVar2 */
