Циклы

 Как и в любом другом командном языке в нашем имеется возможность сократить большую часть работы при помощи организации многократного исполнения набора инструкций, именуемых как циклы.
 Для того, чтобы отличать код внутри цикла, используется фигурные скобки '{ }'.
 
 При создании 'for' циклов было принято решение использовать знакомый всем синтаксис 'for [ item in list ] {...}', где "item" - это элемент из списка, массива или любого другого объекта, унаследовавшего класс 'list'. 

 Также есть еще один тип цикла и его небольшое изменение - это 'while [ <условие продолжения цикла> ]{ ... }' 
 и 'do { ... } while [ <условие продолжения цикла> ]'
