Общее представление:
Строки в командном языке представляют из себя последовательность символов (TODO: уточнить поддержку Юникода).
Строка задаётся последовательностью символов, заключённых между одинарными ('') или двойными ("") кавычками. Если есть необходимость использовать сам символ кавычки в строке, используется обратная косая черта.
Символ обратной косой черты (\) используется для задания некоторых спецсимволов, подробнее ниже.
Спецсимволы:
"\\" — обратная косая черта
"\n" — символ переноса строки
"\b" — символ забоя
"\f" — символ перевода страницы
"\r" — символ возврата каретки
"\t" — символ горизонтальной табуляции
"\v" — символ вертикальной табуляции
"\0" — символ Null
